Home Page Template

This is the template for my home page!

Description

To begin, set up your basic HTML layout. If you want to add bootstrap, add http://maxcdn.bootstrapcdn.com/bootstrap/3.3.7/css/bootstrap.min.css in a link element, set is as the href and set the rel as stylesheet. It was not added below, but will be used in this template.

				<!DOCTYPE html>

				<html>

				<head>

				</head>

				<body>

				</body>

				</html>

Next we will create the menu bar. It's very simple and basic, and doesn't take attention away from the page content.

				<!DOCTYPE html>

				<html>

				<head>

				</head>

				<body>

					  <ul>

						    <!--For this example, we will use three links-->

						    <li><a href="#">Link 1</a></li>

						    <li><a href="#">Link 2</a></li>

						    <li><a href="#">Link 3</a</li>

						  </ul>
	
				</body>

				</html>

This creates our basic HTML menu bar. Now we have to style. You can do this in a separate stylesheet or use style tags, which is what I use. If you choose to use the style tags, put your CSS in the header.

					<style>

					ul {

    					list-style-type: none;

    					margin: 0;

    					padding: 0;

    					overflow: hidden;

    					background-color: #D9D9D9;

					}

					li {

    					float: left;

					}

					li a {

    					display: block;

    					color: black;

    					text-align: center;

    					padding: 14px 16px;

    					text-decoration: none;

					}

					li a:hover {

    					background-color: #EFEFEF;

    					text-decoration: none;

					}

					</style>

Now we are finished with the menu bar! Let's move on to more important things. We will now create the jumbotron element. Add the following HTML below your menu code.

					 <div class="jumbotron">

    					  <h1 align="center">Welcome</h1>

  					</div>

Simple right? Let's style our jumbotron element. Add the following below your menu CSS.

					.jumbotron{

  					  width: 100%;

  					  background: rgba(35, 50, 55, 0.1);

  					  color: white;

					}

This sets the background of the jumbotron to white and makes it transparent. To alter the opacity, increase the decimal value.

Next we will create the three box elements below the jumbotron. Add the following code below the jumbotron code.

					<div class="container">

					  <div class="row">

					    <div class="col-sm-4">

					    </div>
					    <div class="col-sm-4">

					    </div>
					    <div class="col-sm-4">

					    </div>

					  </div>

					</div>

The content will go between the col-sm-4 classes. The container, row, and col elements are part of bootstrap!

Now let's style our boxes. Add the following to your CSS(it's not much at all!)

					.col-sm-4{

					  background-color: #d9d9d9;

					}

You can put your own content in the boxes. I used h3 and p elements in mine.

Next, we will create the loader. Add the following code to your opening body tag. You can use your own background image, or use mine.

					<body style="background-image: url(https://wallpaperscraft.com/image/city_night_church_lights_beauty_58859_1920x1080.jpg); background-size: cover" onload="myFunction()" style="margin:0;">

Add the following under your body tag. Put the closing </div> tag just above your </body> tag.

					<div id="loader">

					</div>

					<div style="display:none;" id="myDiv" class="animate-bottom">

Just before the closing </body> tag, after the closing </div> tag, add the following script.

					<script>

					var myVar;


					function myFunction() {

    					  myVar = setTimeout(showPage, 3000);

					}


					function showPage() {

  						  document.getElementById("loader").style.display = "none";

  						  document.getElementById("myDiv").style.display = "block";

					}

					</script>

Now let's style the loader. Add the following to your CSS.

					#loader {

  						  position: absolute;

  						  left: 50%;

  						  top: 50%;

  						  z-index: 1;

  						  width: 150px;

  						  height: 150px;

 						  margin: -75px 0 0 -75px;

  						  border: 16px solid #f3f3f3;

  						  border-radius: 50%;

  						  border-top: 16px solid #3498db;

  						  border-bottom: 16px solid #F8820B;

  						  width: 120px;

  						  height: 120px;

  						  -webkit-animation: spin 2s linear infinite;

  						  animation: spin 2s linear infinite;

						}


					@-webkit-keyframes spin {

  					  0% { -webkit-transform: rotate(0deg); }

  					  100% { -webkit-transform: rotate(360deg); }

					}


					@keyframes spin {

  					  0% { transform: rotate(0deg); }

  					  100% { transform: rotate(360deg); }

					}


					/* Add animation to "page content" */

					.animate-bottom {

  					  position: relative;

  					  -webkit-animation-name: animatebottom;

  					  -webkit-animation-duration: 1s;

  					  animation-name: animatebottom;

  					  animation-duration: 1s

					}


					@-webkit-keyframes animatebottom {

  					  from { bottom:-100px; opacity:0 } 

  					  to { bottom:0px; opacity:1 }

					}


					@keyframes animatebottom { 

  					  from{ bottom:-100px; opacity:0 } 

  					  to{ bottom:0; opacity:1 }

					}


					#myDiv {

  					  display: none;

  					  text-align: center;

					}

All done! Now we have the basic page template, with the loader!
