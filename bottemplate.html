This is just an example of a basic bot setup that uses discord.js. Please note that you must have node.js in order to run discord bots. You must also have a token and authorization. For help deploying the bot, please see this guide.

Code

var Discord = require("discord.js");

			var bot = new Discord.Client();

			bot.on("ready", ()=> {

			bot.user.setGame("namegoeshere")

			});

			bot.on("message", msg => {

			  if(msg.content == "textgoeshere"){

				    bot.channel.sendMessage("textgoeshere");
			
				  } else if (msg.content == "textgoeshere"){

				    bot.channel.sendMessage("textgoeshere");

			  }

			});

			bot.login(tokengoeshere);

This is a basic setup. If you want to add more text commands, just add more else if statements.

Basic Terms: if(msg.content == "textgoeshere") reads a message and if it matches textgoeshere, will evaluate the if statement. If not, it will move on to the else if statement. If it still does not match, then the bot will not post anything. For example, if textgoeshere(in the if statement) was set to "Hello", and a message was sent that says "Hello", then the if statement will evaluate.
